@layerdraw/engine-wasm third-party notices

The final release artifact co-distributes the unmodified wasm_exec.js support file
from the exact pinned Go toolchain used to build layerdraw-engine.wasm. That file
and the Go runtime linked into the WebAssembly binary are provided under the Go
project's BSD 3-Clause license. The integration build must retain the upstream
copyright and license text, verify the support-file digest, and enumerate the
runtime closure in the artifact-specific CycloneDX SBOM.

This TypeScript transport shell has no runtime npm dependencies. Playwright,
TypeScript, and Node type declarations are development/test inputs only and are
not runtime components of the published Engine WASM artifact.
