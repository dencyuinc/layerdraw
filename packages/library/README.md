# `@layerdraw/library`

Framework-neutral Library view model for F49–F53. Applications inject a
`RegistryClient`; React or another shell renders snapshots and invokes intents.
Every trust, dependency, compatibility, archive, impact, and transaction
decision displayed here is an opaque host result. The model never inspects an
archive, rewrites LDL, resolves a dependency, or mutates a package tree.
